# Roadmap

## Phase 1 — MVP

Goal: publish a small English-language website for Pro Anima cat shelter volunteers.

Features:

- Home page
- Cat catalogue
- Cat profile page
- Volunteer guide
- Read-only cat visit calendar
- Manual availability updates
- Basic tests
- Production deploy

## Phase 2 — Better content management

- Store data outside the source code
- Add JSON or database-backed data source
- Improve content editing process
- Add better photo handling

## Phase 3 — Dog volunteering support

- Add dog volunteering direction to the visit model
- Add UI filter or tabs for cats and dogs
- Add dog-specific volunteer guide
- Add dog visit calendar entries

## Phase 4 — Online registration

- Register for a visit through the website
- Cancel registration
- Track free places automatically
- Add waitlist
- Send notifications to coordinator

## Phase 5 — Telegram integration

- Sync selected visit information with volunteer chats
- Notify about free places
- Send reminders
- Support coordinator commands
